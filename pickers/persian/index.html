Other pickers

Updated Tue 6 Sep 2016 • tags arabic, pickers, apps.

This Unicode character picker allows you to produce or analyse runs of Persian text using the Arabic script. Character pickers are especially useful for people who don't know a script well, as characters are displayed in ways that aid identification.

To properly display the text you will need to choose a font that is loaded on your system or device. The font list indicates which fonts are standard for Mac (Snow Leopard/Lion) and Windows7. See more information about standard OS fonts in Mac and Windows.

To produce characters in the text area, click on character shapes, or use your keyboard for Latin characters, delete, etc. Then cut & paste the result to your document, or use the controls to get further information about the characters. You must have JavaScript enabled.

Mobile devices
When working on an iPad or similar device, you should turn off more controls/Autofocus. This prevents the keyboard popping up after you input every character. You may also need to select a character twice to add it to the output field.
About the chart
Includes characters in the Unicode Arabic block that are used for writing modern Standard Persian. This is an early version of the picker for Persian. It is likely to change a little in the future as the required set of characters is refined. (It currently has some characters that are not needed, and may be missing some that are.)
